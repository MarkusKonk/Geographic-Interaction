Gesture & Speech Map

IWGI - Interacting with Geographic Information

This is a project of Masterstudents from the field of Geoinformatics of the WWU in Münster. In that one a map was created, which can be used by speech and gestures. For the gesture recognition a Kinect device is necessary. Within the help page the basic gestures and commands are explained.

Required components for using the map:

  • Kinect device
  • Gesture recognition app (can be find in "Download App")
Recommended components for using the map:
  • Microphone
