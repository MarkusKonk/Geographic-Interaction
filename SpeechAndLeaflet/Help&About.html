What is this System?

This system is a multi-modal application. This application allows the users to control the map components by Gesture and Speech commands with the help of Keyboard and Mouse. To use this application you need Kinect device to detect user gestures, a microphone to detect use speech and an internet connection.

System Data Flow Architecture



Start the system

- Make sure that the Kinect device is plug-in and the Microphone is on
- Stand in front of the Kinect device
- Wave your RIGHT hand until the right hand indicator at the wabpage header become green
- Wave your LEFT hand until the left hand indicator at the wabpage header become green
- The Speech recogniser is automatically working

Navigate the map

You can navigate the map using the gestures AND/OR speech commands
Gestures :
- To use the gestures make sure that the Left and Hand indicators at the webpage header are green.
- Grap your left hand (keep grapping throw navigation)
- Move your right hand to move the map
Speech Commands:
- To move-up the map; Say : "Up"
- To move-down the map; Say : "Down"
- To move the map to the left; Say : "Left"
- To move the map to the right; Say : "Right"
Note: you can use the gestures and speech commands at the same time.

Zoom In and Out

Zoom In


- Perform the Zoom In Gesture : OR
- Say : "Zoom in"
Note: In both cases a message "Zoom In" will appear

Zoom Out


- Perform the Zoom Out Gesture : OR
- Say : "Zoom out"
Note: In both cases a message "Zoom out" will appear

Mark a point (Marker)

- Perform the "Clicking" gesture using your right or left hand: OR
- Say : "point"
- In both cases a side bar will appear, Then
-Enter the point name by speech or using the keyboard
-Perform the "Tab" gesture to move to description field
-Enter the point description by speech or by using the keyboard

Draw a Road

- To start drawing a road ; say : "Road"
-Perform the "Clicking" gesture to assign points

Enable or Disable MiniMap

Enable MiniMap


- Say : "Enable MiniMap"

Disable MiinMap


- Say : "Disable MiniMap"

Measurement

- To start the measurement tool; say : "Start Measurement"
- Perform the "Clicking" gesture to assign points
- To stop the measurement tool; Say : "Stop Measurement"

What are the system requirements?

Required components for using the map:
  • Kinect device. Need to install OpenNI: Open-source SDK for 3D sensors
  • Microphone

How can I know that the system is tracking my hands gestures?

There are two indicators at the page navigation header, one is for the left hand and the other is for the right hand. Green color mean that the hand is recognized and the Red color mean the hand is not recognised. If one of the hands is not recognised you need to re-wave the unrecognised hand until the indicator convert to green
